## 🚀 Getting Started

1. **Clone the repository:**

```bash
git clone https://github.com/CoffeAddict/My-Portfolio.git
```

2. **Navigate to the project directory:**

```bash
cd my-portfolio
```

3. **Install Dependencies:**

```bash
npm ci
```

4. **Start the development server:**

```bash
npm run dev
```
